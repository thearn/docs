scipy.special.pbwa¶

scipy.special.pbwa(a, x) = <ufunc 'pbwa'>¶

Parabolic cylinder function W.

The function is a particular solution to the differential equation

\[y'' + \left(\frac{1}{4}x^2 - a\right)y = 0,\]

for a full definition see section 12.14 in [R526].

Parameters:

a : array_like

Real parameter

x : array_like

Real argument
Returns:

w : scalar or ndarray

Value of the function

wp : scalar or ndarray

Value of the derivative in x

Notes

The function is a wrapper for a Fortran routine by Zhang and Jin [R527]. The implementation is accurate only for |a|, |x| < 5 and returns NaN outside that range.

References
